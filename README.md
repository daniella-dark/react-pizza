Интернет-магазин пиццерии на ReactJS.
***

Стек технологий
----------------
* ReactJS + React Hooks
* TypeScript
* React Router
* Redux Toolkit
* Axios
* SCSS / CSS Modules
* React Content Loader
* React Paginate
* Lodash.Debounce

Установка и запуск
---------
Установка зависимостей:\
`npm install`

Сборка проекта в режиме разработки и запуск локального сервера:\
`npm start`
